This project uses JSON server. To run the project, follow these steps:

1. Clone the repository using 'git clone' command in Git;

2. Open a new terminal and navigate to the directory where you cloned the repository;

3. Install the dependencies by running 'npm install';

4. Open new terminal to start the JSON server and type 'npm run json-server;

5. In a new terminal, start the React project by running npm start.
